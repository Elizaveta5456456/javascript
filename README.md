Что реализовано

- Главная страница
- Страница меню с фильтрацией
- Добавление в корзину и хранение в localStorage
- Страница оформления заказа с валидацией
- Подтверждение заказа
- Навигация через Vue Router
- Анимация и стили

Как запустить

```bash
npm install
npm run dev
```
